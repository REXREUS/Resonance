## How to Take Screenshots

### On Android Device
1. Navigate to the screen you want to capture
2. Press **Power + Volume Down** simultaneously
3. Screenshot will be saved to your gallery


### Using ADB
```bash
adb exec-out screencap -p > screenshot.png
```

## Tips for Good Screenshots

1. **Use a clean device** - Hide notification bar clutter
2. **Show realistic data** - Complete a few sessions first
3. **Consistent theme** - Use dark mode for all screenshots
4. **High resolution** - Use a device with 1080p+ display
5. **No personal info** - Blur or remove any sensitive data
